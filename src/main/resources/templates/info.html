Wykorzystane technologie!

  • HTML 5 - Wszystkie pliki w katalogu template zostały napisane używając języka HTML w wersji 5.
  • CSS - w pliku main.css w katalogu /static/css znajdują się zdefiniowane arkusze stylów.
  • Thymeleaf - silnik szablonów, narzędzie bardzo często wykorzystywane do współpracy z frameworkiem Spring.
  • Bootstrap - biblioteka graficzna określająca styl wszystkich nieopisanych komponentów.
  • SpringBoot - wersja 2.4.3, framework projektu od strony backendu, biblioteka oparta na javie będąca sercem aplikacji.
  • SpringSecurity - ochrona springa, która uniemożliwia dostanie się do niepożądanych treści przez niezalogowanego użytkownika.
  • Java w wersji 11, jeden z najpopularniejszych języków programowania.
  • Baza danych H2, umożliwia trzymanie danych w pamięci aplikacji.
  • JQuery i JavaScript - w pliku main.js w katalogu /static/js zdefiniowane są skrypty wykonywane po stronie klienta.
  • GIT - system wersjonowania projektu.
  • BitBucket - repozytorium projektu, prowadzący otrzyma dostęp do kodu źródłowego projektu.

Spełnienie wymagań projektu
  • Strona aplikacja spełnia wymagania projektu ponieważ zostały spełnione wszystkie podpunkty z wymagań.
  • Aplikacja działa poprawnie we wszystkich popularnych przeglądarkach, została sprawdzona w Chrome, Mozilla Firefox, Opera, Internet Explorer.
  • Zawiera alternatywny tekst dla obrazków. Znajdują się one głównie w liście dostępnych pojazdów do wypożyczenia.
  • Poza HTML i CSS wykorzystuje wiele innych technologii, między innymi Java, Spring, H2 Database, JavaScript, JQuery, SpringSecurity, GIT, BitBucket.
  • Na stronie głównej oraz w widoku pojazdów znajdują się animacje napisane w CSS.
  • Przedewszystkim serwis był przewidziany do możliwości prowadzenia wypożyczalni samochodów, strona umożliwia dodawanie i usuwanie pojazdów, a także ich wypożyczanie oraz przeglądanie historii wypożyczeń.
  • Strona jest intuicyjna, pozwala użytkowniki w łatwy i szybki sposób.
